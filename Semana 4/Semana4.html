El procedimiento que debes seguir para cambiar el estilo de presentación de la página web previamente diseñada es el siguiente:

  1. Seleccionar el elemento que queremos cambiar.
  2. Aplicar la propiedad CSS que queremos cambiar.
  3. Asignarle un valor a la propiedad CSS que queremos cambiar.

Por ejemplo, si queremos cambiar el color de fondo de la página web, el código sería el siguiente:

        
            body {
                background-color: #000000;
            }

Si queremos cambiar el color de fondo de un párrafo, el código sería el siguiente:

        
            p {
                background-color: #000000;
            }
